# Summarization Prompts

A collection of reusable prompts for summarizing content efficiently using AI.

---

## 1. Simple Summary

**Use Case:** Quick understanding of content  

**Prompt:**
```text
Act as an Summary Writer.

Summarize the following text in simple and clear language:

[TEXT]
```
**Inputs:**
- Text

**Output:**
- Simple summary

**Variations:**
- Ultra-short summary
- Beginner-friendly summary

## 2. Bullet Point Summary

**Use Case:** Extracting key points

**Prompt:**
```text
Act as an Summary Writer.

Summarize the following content into 5–7 key bullet points:

[TEXT]
```
**Inputs:**
- Text

**Output:**
- Bullet points

**Variations:**
- 3 key points
- Detailed bullet summary

## 3. Detailed Structured Summary

**Use Case:** In-depth understanding

**Prompt:**
```text
Act as an Summary Writer.

Provide a detailed structured summary of the following content:

[TEXT]

Include:
- Main idea  
- Key points  
- Supporting details  
```
**Inputs:**
- Text

**Output:**
- Structured summary

**Variations:**
- Academic style
- Professional report

## 4. One-Line Summary

**Use Case:** Quick overview

**Prompt:**
```text
Act as an Summary Writer.

Summarize the following content in one sentence:

[TEXT]
```
**Inputs:**
- Text

**Output:**
- Single sentence summary

**Variations:**
- Tagline style
- Headline style

## 5. Explain Like I am 5

**Use Case:** Simplifying complex topics

**Prompt:**
```text
Act as an Summary Writer.

Explain the following topic in a very simple way like I am 5 years old:

[TOPIC]
```
**Inputs:**
- Topic

**Output:**
- Simple explanation

**Variations:**
- For beginners
- With examples

**Variations:**

## 8. Comparison Summary

**Use Case:** Comparing two topics

**Prompt:**
```text
Act as an Summary Writer.

Compare and summarize the differences between [A] and [B].

Include:
- Key differences  
- Similarities  
- Conclusion  
```
**Inputs:**
- Topic A
- Topic B

**Output:**
- Comparison summary

**Variations:**
- Table format
- Bullet format

**Inputs:**

**Output:**
- Summary + action steps

**Variations:**

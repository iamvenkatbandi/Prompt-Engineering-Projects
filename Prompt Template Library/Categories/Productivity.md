# ⚡ Productivity & Personal Use Prompts

A curated collection of reusable prompt templates for improving productivity, planning, decision-making, and personal growth using AI.

---

## 1. Daily Planner

**Use Case:** Planning daily tasks

**Prompt:**

```text id="dp72sk"
Create a daily plan for [GOAL].

Include:
- Top priorities  
- Time blocks  
- Breaks  
- End-of-day review  

Tone: Structured and practical
```

**Inputs:**

* Goal

**Output:**

* Daily schedule

**Variations:**

* Student plan
* Workday plan

# ❓ Q&A and Learning Prompts

A curated collection of reusable prompt templates for learning, concept understanding, and knowledge extraction using AI.

---

## 1. Concept Explanation

**Use Case:** Understanding a topic clearly

**Prompt:**

```text
Explain [TOPIC] in simple terms.

Include:
- Definition  
- Key concepts  
- Examples  

Tone: Easy to understand
```

**Inputs:**

* Topic

**Output:**

* Clear explanation

**Variations:**

* Beginner explanation
* Advanced explanation

---

## 2. Step-by-Step Learning

**Use Case:** Learning a new skill or topic

**Prompt:**

```text
Teach me [TOPIC] step-by-step from beginner to advanced.

Include:
- Basic concepts  
- Intermediate topics  
- Advanced concepts  
- Learning roadmap  
```

**Inputs:**

* Topic

**Output:**

* Structured learning guide

**Variations:**

* 7-day learning plan
* 30-day roadmap
